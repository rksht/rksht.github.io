What is a total derivative

Posted on May 5, 2020

Linear approximation

We will only talk about scalar-valued functions.

First we learn about linear approximation - what it exactly is.

Look at a single-variable function $f : R \rightarrow R$. The linear approximation of this function at the point $x = a$, is defined as the function $L : R \rightarrow R$ such that

$L(x) = f(a) + f’(a) * (x - a)$

Linear part is $A(x) = f’(a)x$. So we could write it as $L(x) = f(a) + A(x - a)$

From this we see that linear approximation is just the first two terms of the Taylor series for the function. And you can see that $L$ is a linear function (well, affine really).

Now, what does it mean? We have defined it constructively. But is there another way to define “linear approximation” by some property instead of directly giving a function definition (in this case, that RHS expression for $L$)?

There is.

A function $f : R^n \rightarrow R$ has a total derivative at a point $x = \vec{c}$ if,

$$ \exists A ∈ Linear(R^n \rightarrow R) $$

such that

$$ L(x⃗, c⃗) = f(c⃗) + A(x⃗ - c⃗) \Longrightarrow \lim_{x⃗ \rightarrow c⃗} \frac{f(x⃗) - L(x⃗, c⃗)}{\left\lVert x⃗ - c⃗ \right\rVert} = 0 $$

Here $x⃗ \rightarrow c⃗$ means “over all paths” from $x⃗$ to $c⃗$.

The issue is that there is no general way to get a linear approximation, it depends on the function $f$.

For scalar-valued functions, if linear approximation is defined and the gradient is not $\vec{0}$, the product operator in A becomes a matrix multiplication, and $f’(a)$ is replaced with the gradient vector at $\vec{a}^\intercal$ (it is transposed, so the multiplication is actually doing a dot prod).

But wait. Why can’t you use the gradient vector and say that $A = \nabla f$?

That is because your idea of the gradient’s existence is not correct, bruh.

There’s something called conditional equality.

Coordinates of the gradient if it exists

First, we need the definition of the gradient. Turns out, the function $A$ above, if it does satisfy the linear approximation requirement, is the gradient. That’s pretty much the definition of gradient.

It’s the function, if it exists, can ensure that we can linearly approximate $f$ and as per definition $f$ is differentiable, i.e. we can get a total derivative.

If the gradient exists at a point, it’s coordinates will be the partial derivatives of $f$ at that point. This is what it means to

$$ \nabla f = \langle \frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots \frac{\partial f}{\partial x_n} \rangle $$

Again, $\nabla f$ is not defined as the right hand side vector valued function.

TL;DR

Gradient Exists $\Leftrightarrow$ Total differentiable

Gradient Exists $\Rightarrow$ Gradient is a vector of partial derivatives
