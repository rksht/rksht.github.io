Derivation an RNG following the cosine distribution

Posted on Jan 20, 2020

Using the inversion method.

The CDF is

$$ G(\theta, \phi) = \frac{\sin^2\theta}{2\pi} (\phi+\pi) $$

We got two random variables that we need to sample $\theta$ and $\phi$.

First, as per the technique, we compute $G_{max} ≡ G(\theta, \phi_{max})$ . Noting that $\phi_{max} = \pi$ we compute $G(\theta,\pi) = \sin^2\theta$.

Now, if we generate a canonical random variable $r_j$, let $r_j = G(\theta_j,\pi)$, where $\theta_j$ stands for the sampled value of $\theta$. I’m using $j$ to denote the $j$-th sample value. No other reason.

Which gives, $\theta_j = \arcsin(\sqrt{r_j})$.

Onto computing $\phi$. After we generate a random variable $s_j$, we equate

$$ s_j = \frac{G(\theta_j, \phi)}{G(\theta_j, \phi_{max})} $$

Doing so yields, $\phi = 2\pi\space s_j - \pi$. Shirley and other sources use $[0, 2\pi]$ as the range of $\phi$. I am using $[-\pi, \pi]$ as the range. Hence that $-\pi$ term. I would recommend just using $\phi = 2\pi\space s_j$ and the $[0,2\pi]$ range.
